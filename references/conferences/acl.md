# ACL — Annual Meeting of the Association for Computational Linguistics

## Basic Info
- Full name: Annual Meeting of the Association for Computational Linguistics
- CCF Level: A
- Acceptance rate: ~22% (2024: ~21.3%)
- Page limit: 8 pages + supplementary
- Deadline: Typically February (notification May)
- Double-blind review: Yes

## Review Weights
- Innovation: 25% — Linguistic insight valued
- Theory: 20% — Connection to linguistics preferred
- Empirical: 30% — Strong NLP evaluation required
- Clarity: 15% — Must connect to linguistic motivation
- Reproducibility: 10% — Code release expected

## Writing Style
- Title: Task-oriented, linguistic connection visible
- Abstract: Linguistic motivation → NLP task → method → results
- Introduction: Connect to linguistic theory, not just engineering
- Experiments: NLP benchmarks, error analysis required

## Recent Trends (2024-2026)
- LLM reasoning and chain-of-thought
- Multilingual NLP
- Interpretability and analysis
- NLP for social good
- Efficient NLP methods

## Reviewer Common Concerns
- "No linguistic insight — this is just engineering"
- "Where's the connection to linguistic theory?"
- "Missing error analysis"
- "Only benchmark numbers, no qualitative analysis"

## Anti-patterns
- Pure engineering without linguistic basis
- Chasing SOTA without understanding
- No error analysis or linguistic analysis
- Ignoring linguistic phenomena

## Success Patterns
- Linguistic motivation + NLP performance
- Error analysis that reveals linguistic patterns
- Method grounded in linguistic theory
- Qualitative analysis that provides insight
