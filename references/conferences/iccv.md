# ICCV — IEEE/CVF International Conference on Computer Vision

## Basic Info
- Full name: IEEE/CVF International Conference on Computer Vision
- CCF Level: A
- Acceptance rate: ~26% (2024: ~26%)
- Page limit: 8 pages + supplementary
- Deadline: Typically March (notification July)
- Double-blind review: Yes

## Review Weights
- Innovation: 30% — Technical depth valued
- Empirical: 30% — Design rationale for every component
- Theory: 15% — Not primary but helpful
- Clarity: 15% — Method details must be clear
- Reproducibility: 10% — Code release expected

## Writing Style
- Title: Emphasize technical contribution
- Abstract: Technical challenge → method insight → results
- Introduction: Technical motivation first, then visual results
- Experiments: Principled design with clear ablations for each component

## Recent Trends (2024-2026)
- 3D vision and reconstruction
- Embodied AI
- Open-vocabulary recognition
- Efficient vision models
- Vision foundation models

## Reviewer Common Concerns
- "Design choices unjustified"
- "Why this specific architecture/component?"
- "End-to-end without component analysis"
- "No insight into why the method works"

## Anti-patterns
- End-to-end black box without component analysis
- Unjustified design choices
- Missing ablation for individual components
- No analysis of what each component contributes

## Success Patterns
- Principled design with clear ablations
- Every component justified with analysis
- Technical depth with visual evidence
- Clear explanation of design rationale
