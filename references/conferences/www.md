# WWW — The Web Conference

## Basic Info
- Full name: The Web Conference (formerly International World Wide Web Conference)
- CCF Level: A
- Acceptance rate: ~15% (2024: ~15%)
- Page limit: 10 pages + supplementary
- Deadline: Typically October (notification January)
- Double-blind review: Yes

## Review Weights
- Innovation: 20% — Web-scale systems contribution valued
- Empirical: 30% — Web-scale validation required
- Systems: 25% — Scalability and system design matter
- Theory: 10% — Helpful but not primary
- Clarity: 15% — Must address web-specific challenges

## Writing Style
- Title: Web-scale or web-specific angle visible
- Abstract: Web problem → system challenge → method → web-scale results
- Introduction: Start with web-scale motivation
- Experiments: Web-scale datasets, system benchmarks, scalability analysis

## Recent Trends (2024-2026)
- Web3 and decentralized web
- Social media analysis and misinformation
- Web security and privacy
- LLM for web applications
- Web-scale recommendation systems

## Reviewer Common Concerns
- "Doesn't address web-scale challenges"
- "Small-scale lab experiment — where's the web validation?"
- "Not specific to the web"
- "Scalability not demonstrated"

## Anti-patterns
- Small-scale lab experiment
- Not addressing web-specific challenges
- No scalability analysis
- Generic algorithm without web context

## Success Patterns
- Web-scale validation + system contribution
- Scalable system with real web data
- Novel web-specific challenge with principled solution
- System contribution + web application impact
