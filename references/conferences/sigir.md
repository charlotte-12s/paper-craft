# SIGIR — ACM SIGIR Conference on Research and Development in Information Retrieval

## Basic Info
- Full name: ACM SIGIR Conference on Research and Development in Information Retrieval
- CCF Level: A
- Acceptance rate: ~16% (2024: ~16%)
- Page limit: 8 pages + supplementary
- Deadline: Typically January (notification April)
- Double-blind review: Yes

## Review Weights
- Innovation: 25% — User-centric evaluation required
- Empirical: 30% — Beyond offline metrics
- Theory: 15% — IR theory valued
- User Study: 20% — User perspective essential
- Clarity: 10% — Must connect to IR fundamentals

## Writing Style
- Title: IR task-oriented, user perspective visible
- Abstract: IR problem → method → both offline and user metrics
- Introduction: User-centered motivation, not just algorithm-centered
- Experiments: Offline metrics + user study/simulation

## Recent Trends (2024-2026)
- LLM for search and retrieval
- Conversational IR
- Fairness and bias in ranking
- Dense retrieval methods
- Multimodal search

## Reviewer Common Concerns
- "Only offline metrics, no user perspective"
- "NDCG-only without user analysis"
- "No consideration of user experience"
- "Algorithmic novelty unclear"

## Anti-patterns
- NDCG-only without user analysis
- No user-centric evaluation
- Pure algorithm without IR context
- Ignoring fairness and bias

## Success Patterns
- Offline + user study + practical impact
- Algorithmic innovation with user validation
- Fair ranking with bias analysis
- Practical IR system with deployed evaluation
