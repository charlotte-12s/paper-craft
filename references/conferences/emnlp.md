# EMNLP — Conference on Empirical Methods in Natural Language Processing

## Basic Info
- Full name: Conference on Empirical Methods in Natural Language Processing
- CCF Level: A
- Acceptance rate: ~23% (2024: ~22%)
- Page limit: 8 pages + supplementary
- Deadline: Typically June (notification September)
- Double-blind review: Yes

## Review Weights
- Innovation: 20% — Empirical rigor most valued
- Empirical: 40% — Reproducibility and statistical significance required
- Theory: 10% — Not primary criterion
- Clarity: 15% — Must report all experimental details
- Analysis: 15% — Error analysis and insights

## Writing Style
- Title: Specific to NLP task and method
- Abstract: Task → method → detailed results with significance
- Introduction: Practical NLP problem → empirical finding → method
- Experiments: Statistical significance, error analysis, reproducibility details

## Recent Trends (2024-2026)
- LLM evaluation and benchmarking
- Data contamination and cleaning
- Annotation quality and disagreement
- Prompting and in-context learning
- Multilingual and cross-lingual NLP

## Reviewer Common Concerns
- "Missing error analysis"
- "No statistical significance testing"
- "Not reproducible — missing details"
- "SOTA-chasing without understanding"

## Anti-patterns
- SOTA-chasing without understanding why
- No error analysis
- Missing statistical significance
- Incomplete reproducibility details

## Success Patterns
- Thorough analysis + reproducible + insightful errors
- Comprehensive error analysis with linguistic insights
- Statistical rigor with multiple significance tests
- Reproducible pipeline with released code and data
