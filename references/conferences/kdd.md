# KDD — ACM SIGKDD Conference on Knowledge Discovery and Data Mining

## Basic Info
- Full name: ACM SIGKDD Conference on Knowledge Discovery and Data Mining
- CCF Level: A
- Acceptance rate: ~15% (2024: ~15%)
- Page limit: 9 pages + appendix
- Deadline: Typically February (notification May)
- Double-blind review: Yes

## Review Weights
- Innovation: 20% — Real-world impact paramount
- Empirical: 35% — Must validate on real data
- Significance: 25% — Deployment story valued
- Theory: 10% — Helpful but not required
- Clarity: 10% — Must explain practical value

## Writing Style
- Title: Problem-driven, emphasize real-world application
- Abstract: Real-world problem → data challenge → method → practical impact
- Introduction: Start with business/real-world motivation
- Experiments: Real data + deployment story, not just benchmarks

## Recent Trends (2024-2026)
- Graph neural networks for real applications
- LLM + tabular data integration
- Responsible AI and fairness
- Automated ML and AutoML
- Time-series and spatiotemporal mining

## Reviewer Common Concerns
- "No real-world validation"
- "Only synthetic data experiments"
- "Missing deployment story"
- "Scalability not demonstrated"

## Anti-patterns
- Synthetic-only experiments
- No real-world validation
- Algorithm without practical application
- Small-scale experiments claiming scalability

## Success Patterns
- Real data + deployment + business impact
- Scalable algorithm validated on large real datasets
- Deployment story with practical outcomes
- Methods that solve real data mining challenges
