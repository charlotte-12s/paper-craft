# CVPR — IEEE/CVF Conference on Computer Vision and Pattern Recognition

## Basic Info
- Full name: IEEE/CVF Conference on Computer Vision and Pattern Recognition
- CCF Level: A
- Acceptance rate: ~25% (2024: ~23.6%)
- Page limit: 8 pages + supplementary
- Deadline: Typically November (notification February)
- Double-blind review: Yes

## Review Weights
- Innovation: 25% — Visual results matter most
- Empirical: 35% — Strong visual evidence + quantitative results
- Theory: 15% — Not primary criterion
- Clarity: 15% — Visual clarity essential
- Demo: 10% — Demo video valued

## Writing Style
- Title: Specific to vision task, method name prominent
- Abstract: Visual problem → visual result (with numbers)
- Introduction: Show visual examples early, hook with impressive results
- Experiments: Visual comparisons required, qualitative + quantitative, demo video bonus

## Recent Trends (2024-2026)
- 3D generation and reconstruction
- Video understanding and generation
- World models
- Open-vocabulary recognition
- Vision-language models

## Reviewer Common Concerns
- "Visual results unconvincing"
- "Only numbers, no visual comparison"
- "Failure cases not shown"
- "Method doesn't generalize to real images"

## Anti-patterns
- Numbers-only without visual comparison
- Cherry-picked visual results
- No failure case analysis
- Comparing against weak/outdated baselines
- Ignoring real-world visual complexity

## Success Patterns
- Striking visual results + SOTA numbers
- Comprehensive visual comparison (side-by-side)
- Clear visual evidence of improvement
- Demo video showing real-time results
