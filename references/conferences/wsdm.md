# WSDM — ACM International Conference on Web Search and Data Mining

## Basic Info
- Full name: ACM International Conference on Web Search and Data Mining
- CCF Level: A
- Acceptance rate: ~15% (2024: ~15%)
- Page limit: 8 pages + supplementary
- Deadline: Typically August (notification November)
- Double-blind review: Yes

## Review Weights
- Innovation: 30% — Algorithmic novelty critical
- Empirical: 30% — Efficiency and improvement over baselines
- Theory: 15% — Algorithmic analysis valued
- Clarity: 15% — Must be clearly algorithmic contribution
- Application: 10% — Search/mining application required

## Writing Style
- Title: Algorithm + search/mining task
- Abstract: Search/mining problem → algorithmic contribution → efficiency + accuracy
- Introduction: Clear algorithmic motivation
- Experiments: Efficiency analysis + search/mining benchmarks

## Recent Trends (2024-2026)
- Dense retrieval methods
- LLM for search and ranking
- Session search and conversational search
- Knowledge graph mining
- Fairness in search and recommendation

## Reviewer Common Concerns
- "Algorithmic contribution unclear"
- "Application-only without algorithmic novelty"
- "No efficiency analysis"
- "Improvement is marginal"

## Anti-patterns
- Application-only without algorithmic novelty
- No efficiency comparison
- Weak baselines
- Marginal improvement without clear algorithmic insight

## Success Patterns
- Novel algorithm + efficiency + search/mining application
- Clear algorithmic innovation with efficiency gains
- Strong baselines with significant improvement
- Both accuracy and efficiency improvements
