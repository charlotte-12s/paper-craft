# S&P — IEEE Symposium on Security and Privacy

## Basic Info
- Full name: IEEE Symposium on Security and Privacy
- CCF Level: A
- Acceptance rate: ~12% (2024: ~12%)
- Page limit: 13 pages + supplementary
- Deadline: Typically September (notification January)
- Double-blind review: No (but anonymized submission)

## Review Weights
- Innovation: 20% — Security rigor is paramount
- Theory: 25% — Formal guarantees and threat models
- Empirical: 25% — Real attacks and defenses
- Practical: 20% — Real-world security impact
- Clarity: 10% — Precise security terminology required

## Writing Style
- Title: Attack/defense + target system
- Abstract: Threat model → attack/defense → formal guarantees → real-world impact
- Introduction: Precise threat model definition, formal security guarantees
- Experiments: Real attacks demonstrated, formal proofs, adaptive attacker evaluation

## Recent Trends (2024-2026)
- LLM security and adversarial attacks
- ML privacy and membership inference
- Supply chain security
- Hardware security
- Post-quantum cryptography

## Reviewer Common Concerns
- "Threat model incomplete"
- "Defense doesn't consider adaptive attacker"
- "No formal security analysis"
- "Attack is not realistic"

## Anti-patterns
- Defense without considering adaptive attacker
- Vague threat model
- No formal security guarantees
- Unrealistic attack assumptions

## Success Patterns
- Strong threat model + formal analysis + real attacks
- Precise threat model with formal guarantees
- Real attacks demonstrated with defense against adaptive adversaries
- Formal proof of security properties
