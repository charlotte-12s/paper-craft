# ICML — International Conference on Machine Learning

## Basic Info
- Full name: International Conference on Machine Learning
- CCF Level: A
- Acceptance rate: ~25% (2024: 26.3%)
- Page limit: 9 pages + unlimited appendix
- Deadline: Typically January (author feedback March, notification April)
- Double-blind review: Yes

## Review Weights
- Innovation: 35% — Novel contributions weighted heavily
- Empirical: 25% — Strong experimental validation expected
- Theory: 25% — Theoretical grounding preferred but not required
- Clarity: 15% — Must be well-written and reproducible

## Writing Style
- Title: Specific and technical, avoid hype words ("groundbreaking", "revolutionary")
- Abstract: Problem → limitation of existing approaches → key insight → method name → results (1-2 key numbers). 150-250 words.
- Introduction: Start with broad motivation, narrow to specific gap, present insight, preview method. 4-6 paragraphs.
- Experiments: At least 3 diverse benchmarks, strong baselines (SOTA within last year), ablations for every design choice, efficiency analysis encouraged.

## Recent Trends (2024-2026)
- Scaling laws and their implications
- Efficient training/inference methods
- Foundation model adaptations
- Multi-modal learning
- Causal reasoning in ML
- AI safety and alignment

## Reviewer Common Concerns
- "How is this different from [similar recent paper]?"
- "The improvement is marginal — is it within statistical variance?"
- "Missing comparison against [specific baseline]"
- "The method seems overly complex for the gains achieved"
- "Limited evaluation — only tested on X domain"

## Anti-patterns
- Stacking 3+ existing techniques without clear justification
- Claiming "SOTA" with only 0.1% improvement on a single benchmark
- Heavy math notation without intuitive explanation
- Comparing against weak/outdated baselines
- No ablation study or only partial ablations

## Success Patterns
- Simple, principled method with clear intuition + solid theory
- Empirical observation → insight → method (observation-driven narrative)
- Unified framework that subsumes multiple existing approaches
- Strong results across diverse settings with thorough ablations
