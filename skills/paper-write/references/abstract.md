# Abstract Writing Guide

## TL;DR Formula

Problem → Gap → Method → Results (4 sentences max)

**Sentence 1 (Problem)**: State the problem and its importance.
**Sentence 2 (Gap)**: What's missing in existing approaches.
**Sentence 3 (Method)**: Your key idea and approach.
**Sentence 4 (Results)**: Main result with 1-2 key numbers.

## Template

> [Problem statement] is important because [motivation]. However, existing approaches [limitation/gap]. We propose [method name], which [key innovation]. Experiments on [benchmarks] show that [method] achieves [key number], outperforming [baseline] by [improvement].

## Length: 150-250 words

## Common Mistakes
- Vague contribution: "We propose a novel method" → What specifically is novel?
- Missing numbers: "Our method achieves better results" → Better by how much?
- Too much background: Don't spend >1 sentence on context
- Over-technical: Abstract should be readable by non-experts
- Multiple contributions listed without clear main one → Lead with the strongest
